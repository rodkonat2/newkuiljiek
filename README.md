# 新酷家电
第六代宝马 M3 已迎来 2027 款车型，其生命周期也将正式迎来终点。宝马方面确认，现款 G80 M3 不会再推出 2028 款车型。

宝马产品规划专家斯科特 · 斯特林在接受《BimmerLife》采访时透露，2027款将成为第六代M3的最后一个年款，具体停产日期仍未确定。

Bimmer Post 论坛有消息称，最后一辆现款 M3 预计将在 2027 年 2 月下线。下一代燃油版 M3 则要等到 2028 年前后才会登场，并首次离开慕尼黑工厂生产。代号 G84 的第七代 M3 未来将转移至丁戈尔芬工厂制造，成为首款不在慕尼黑生产的燃油版 M3。

与此同时，代号 ZA0 的纯电 M3 也已经提上日程，预计明年开始在慕尼黑投产。宝马此前已经多次预告这款新车，新车将采用四电机布局，并加入模拟 V10 发动机声浪和模拟换挡功能。

动力方面，电动版 M3 将搭载 M 专属电池组，净容量超过 100 千瓦时，输出功率也会明显高于 i3 50 xDrive 的 463 马力。

燃油版 M3 将沿用双涡轮增压 3.0 升直列六缸发动机。同时还有消息称，下一代 M3 预计只提供自动变速箱和四轮驱动版本。

据外媒 motor1 今日报道，宝马暂时不会完全放弃传统性能车路线，现款 M2 和 M4 据传仍会持续生产到本世纪 20 年代末。未来数年内，消费者依然可以买到配备手动变速箱的宝马双门后驱的性能车。

宝马 M 部门负责人弗兰克 · 范米尔称，开发全新手动变速箱未来会越来越困难。“未来开发全新变速箱会非常困难，因为市场已经很小，供应商也不太愿意继续投入。我们仍然很喜欢现有的手动变速箱，也计划在未来几年继续保留，但长期来看，尤其进入下一个十年之后，手动挡可能会越来越难继续存在下去。”
